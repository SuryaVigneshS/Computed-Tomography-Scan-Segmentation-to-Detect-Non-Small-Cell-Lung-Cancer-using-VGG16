# Computed-Tomography-Scan-Segmentation-to-Detect-Non-Small-Cell-Lung-Cancer-using-VGG16
This project develops a deep learning CNN model to detect NSCLC from CT scans with 92.46% accuracy. It includes a Streamlit web app for real-time prediction, report generation, and data download. Users can access the system via GitHub and contact support through the built-in form for clinical assistance.
